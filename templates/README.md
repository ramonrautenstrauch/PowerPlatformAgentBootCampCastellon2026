# 📊 Plantillas para Speakers - Power Platform & Agent BootCamp Castellón

Esta carpeta contiene las **plantillas oficiales de presentación** para todos los speakers del **Power Platform & Agent BootCamp Castellón 2026**, organizado por la **Power Platform User Group Castellón**.

## 📅 Sobre el Evento

El Power Platform & Agent BootCamp Castellón 2026 es el evento de referencia sobre Power Platform. ¡Celebra el festival de Power Platform en Castellón!

La **Power Platform** está compuesta por **Power BI** (análisis y visualización de datos), **Power Apps** (desarrollo de aplicaciones low-code), Power Automate (automatización de procesos y flujos de trabajo), **Power Virtual Agents** (creación de chatbots inteligentes) y **Power Pages** (desarrollo de portales web empresariales), integrando datos y procesos para impulsar la transformación digital de las organizaciones.

## 🎯 Instrucciones para Speakers

### ¿Cómo usar la plantilla?

1. **Descarga** el archivo `GPPB2026 Speaker Template CASTELLÓN.pptx`
2. **Personaliza** el contenido con tu sesión
3. **Mantén** el diseño y branding oficial del evento
4. **Guarda** tu presentación con el nombre de tu sesión

### 💡 Recomendaciones

- ✅ Utiliza la plantilla para mantener coherencia visual en todo el evento
- ✅ Respeta los colores y tipografía del branding oficial
- ✅ Incluye el logo de AgentCamp en las diapositivas principales
- ✅ Añade tus datos de contacto y redes sociales al final
