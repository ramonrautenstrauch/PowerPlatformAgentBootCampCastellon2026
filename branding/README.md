# 🎨 Branding y Material Gráfico - Power Platform & Agent BootCamp Castellón

Esta carpeta contiene todo el **material gráfico y de branding** para la promoción y anuncios del **Power Platform & Agent BootCamp Castellón**, organizado por la **Power Platform User Group Castellón**.

## 📅 Sobre el Evento

El Power Platform & Agent BootCamp Castellón 2026 es el evento de referencia sobre Power Platform. ¡Celebra el festival de Power Platform en Castellón!

La **Power Platform** está compuesta por **Power BI** (análisis y visualización de datos), **Power Apps** (desarrollo de aplicaciones low-code), Power Automate (automatización de procesos y flujos de trabajo), **Power Virtual Agents** (creación de chatbots inteligentes) y **Power Pages** (desarrollo de portales web empresariales), integrando datos y procesos para impulsar la transformación digital de las organizaciones.
